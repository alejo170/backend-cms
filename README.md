## CMS (Courses Management System) Backend

![Badge en Desarollo](https://img.shields.io/badge/STATUS-OK-green)

#### Description 📋

The backend of the course management system is designed to facilitate the management and organization of courses. This system provides tools and functionality to manage all aspects of creating, editing, reading, and deleting both students and courses.

#### Characteristics 💼

- `Creation of courses`: entering the name of the course and enrolling the students.
- `Creation of students`: entering their firstname, lastname, email and age.
- `Register`: users can register with their username and password
- `Login`: users can enter the system with their username and password
- `Logout`: users can log out of the system.

#### Used Technologies 🔧

- Node JS
- Express
- Mongo DB

#### Requirements 🛠️

- Have installed Node JS
- Have a database administrator installed for MongoDB

#### Starting 🚀

_Clone the repo_

```
git clone https://github.com/alejo170/backend-cms.git
```

_Install Yarn packages_

```
yarn install
```

_Run the proyect_

```
yarn run dev
```

_Run the test_

```
yarn run test
```
